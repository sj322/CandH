# C&H - 창업동아리

C&H 창업동아리의 팀원 소개 및 활동 포트폴리오 사이트입니다.

## 🚀 배포

이 프로젝트는 GitHub Pages를 통해 자동으로 배포됩니다.

### 배포 방법

1. GitHub 저장소에 코드를 푸시합니다
2. 저장소 Settings > Pages로 이동
3. Source를 `main` 브랜치로 설정
4. 자동으로 `https://[사용자명].github.io/[저장소명]` 주소로 배포됩니다

## 📁 프로젝트 구조

```
CandH/
├── index.html          # 메인 페이지
├── assets/
│   ├── images/        # 이미지 파일 (로고 등)
│   ├── js/
│   │   └── main.js    # JavaScript 파일
│   └── css/           # CSS 파일 (필요시)
└── README.md
```

## 🛠️ 기술 스택

- HTML5
- Tailwind CSS (CDN)
- JavaScript (Vanilla JS)
- Font Awesome (아이콘)
- Google Fonts (Noto Sans KR)

## 📝 커스터마이징

### 팀원 정보 수정

`index.html` 파일에서 팀원 정보를 수정할 수 있습니다:

- 팀원 이름
- 역할/포지션
- 소개 내용
- SNS 링크

### 포트폴리오 항목 추가

`index.html`의 포트폴리오 섹션에 새로운 프로젝트 카드를 추가할 수 있습니다.

### 로고 이미지 추가

`assets/images/` 폴더에 로고 이미지를 추가하고, 필요시 `index.html`에서 참조하도록 수정하세요.

## 🎨 디자인 특징

- 다크 테마 (로고 이미지의 어두운 배경 반영)
- 글래스모피즘 효과
- 부드러운 스크롤 애니메이션
- 반응형 디자인 (모바일, 태블릿, 데스크톱 지원)
